## 🔐 激活码系统

### 技术实现

**算法：** HMAC-SHA256 签名验证  
**格式：** `XXXX-XXXX-XXXX-XXXX-XXXX-XXXX` (24 字符，6 组)  
**编码：** Crockford Base32  
**验证：** 100% 离线验证  

**数据结构：**
```
激活码 = timestamp(4) + random(4) + license_type(1) + device_slots(2) + signature(4)
       = 15 bytes → Base32 → 24 characters
```

**机器码绑定：**
- CPU ID
- 主板序列号
- 硬盘序列号
- 三合一 SHA256 哈希 (前 16 字符)
