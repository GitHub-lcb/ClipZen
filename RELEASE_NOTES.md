# Release Notes - ClipZen v1.0.0

**发布日期：** 2026-03-12  
**版本类型：** 正式发布 🎉

## 🎉 v1.0.0 正式发布！

这是 ClipZen 的第一个正式版本，标志着项目从开发阶段进入生产就绪阶段！

### 💎 Pro 版本系统

v1.0.0 引入了完整的 Pro 版本激活系统，支持淘宝售卖！

#### 新增功能

**激活码系统**
- ✅ 离线激活码验证 (HMAC-SHA256)
- ✅ 机器码绑定 (CPU+ 主板 + 硬盘)
- ✅ 三种许可证类型：
  - 标准版 (¥29.9) - 1 台设备
  - 家庭版 (¥49.9) - 3 台设备
  - 企业版 (¥99) - 5 台设备
- ✅ 激活对话框 UI
- ✅ 设置面板集成 Pro 入口

**Pro 功能**
- 🔐 加密存储保护
- ♾️ 无限历史记录 (免费版 7 天)
- 🎨 自定义主题
- 🏆 终身免费更新
- 💬 优先技术支持
