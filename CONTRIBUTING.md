# Contributing

Start with the repository guide in [`AGENTS.md`](./AGENTS.md). It covers the project layout, build and test commands, code style, and pull request expectations for this repo.

## Before You Send Changes
- Run `npm run lint` and `npm run build` for frontend changes.
- Run `cd src-tauri; cargo test` when touching Rust code.
- Include screenshots or recordings for UI updates.
- Keep commits aligned with the existing Conventional Commits pattern, such as `feat: ...` or `fix: ...`.
