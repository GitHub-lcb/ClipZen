## 快速开始

### 1. 下载与安装

**Windows:**
1. 下载 `ClipZen_0.1.0_x64-setup.exe`
2. 双击运行安装程序
3. 按提示完成安装

**macOS:**
1. 下载 `ClipZen_0.1.0_aarch64.dmg` (Apple Silicon) 或 `ClipZen_0.1.0_x64.dmg` (Intel)
2. 拖动应用到「应用程序」文件夹
3. 首次运行需在「系统偏好设置 → 安全性」中允许

**Linux:**
```bash
# Ubuntu/Debian
sudo dpkg -i ClipZen_0.1.0_amd64.deb

# 或使用 AppImage
chmod +x ClipZen_0.1.0_amd64.AppImage
./ClipZen_0.1.0_amd64.AppImage
```

### 2. 首次启动

1. 启动后，按 `Ctrl+Shift+V` (Mac: `Cmd+Shift+V`) 呼出主界面
2. 主界面会自动显示在屏幕中央
3. 开始复制内容，软件会自动记录

### 3. 基本使用

1. **复制内容** - 正常复制任何文本、图片、文件
2. **呼出界面** - 按 `Ctrl+Shift+V`
3. **选择内容** - 用鼠标或 ↑↓ 键选择
4. **粘贴内容** - 按 `Enter` 或点击内容
5. **关闭界面** - 按 `Esc` 或点击外部

## 激活 Pro 版本

### 获取激活码

购买后您会收到：
- 激活码（格式：`XXXX-XXXX-XXXX-XXXX`）
- 下载链接
- 本用户手册

### 激活步骤

1. 打开小安剪贴板
2. 点击右上角 **⚙️ 设置** 图标
3. 在「Pro 激活」区域点击 **「激活」** 按钮
4. 输入激活码（会自动格式化）
5. 点击 **「立即激活」**
6. 激活成功后会显示「✓ 激活成功」

### 激活状态说明

| 状态 | 说明 |
|-----|------|
| 未激活 | 使用免费版功能 |
| 已激活 - 标准版 | 完整 Pro 功能，1 台设备 |
| 已激活 - 家庭版 | 完整 Pro 功能，3 台设备 |
| 已激活 - 企业版 | 完整 Pro 功能，5 台设备 |

### 更换设备

如需在新电脑上使用：

1. 在旧电脑上打开设置
2. 点击 **「反激活」**
3. 在新电脑上用同一激活码激活

**注意：** 标准版允许 3 次自助重置，超过次数需联系客服。

## 核心功能

### 1. 剪贴板历史

**自动记录：**
- 软件后台自动记录所有复制内容
- 支持文本、图片、文件
- 按时间倒序排列

**查看历史：**
- 按 `Ctrl+Shift+V` 呼出界面
- 滚动查看所有历史记录
- 置顶内容显示在最上方

### 2. 搜索功能

**快速搜索：**
1. 呼出界面后按 `Ctrl+F` 聚焦搜索框
2. 输入关键词
3. 实时过滤结果

**搜索范围：**
- 文本内容（全文搜索）
- 标签
- 文件路径

### 3. 标签管理

**添加标签：**
1. 将鼠标悬停在记录上
2. 点击右下角 **🏷️ 标签** 图标
3. 输入标签名称，按 `Enter` 添加
4. 可选择已有标签

**按标签筛选：**
1. 呼出界面后，顶部显示所有标签
2. 点击标签名称筛选
3. 点击「全部」取消筛选

### 4. 置顶功能

**置顶记录：**
1. 将鼠标悬停在记录上
2. 点击 **📌** 图标
3. 置顶内容显示在最上方

**取消置顶：**
- 再次点击 **📌** 图标

### 5. 加密保护（Pro 功能）

**启用密码保护：**
1. 打开设置
2. 开启「密码保护」
3. 设置密码（至少 6 位）
4. 确认密码

**查看加密内容：**
1. 点击加密记录
2. 输入密码
3. 查看完整内容

### 6. 数据管理

**导出历史：**
1. 打开设置 → 数据管理
2. 点击「导出数据」
3. 选择保存位置
4. 导出为 JSON 格式

**导入历史：**
1. 打开设置 → 数据管理
2. 点击「导入数据」
3. 选择之前导出的 JSON 文件
4. 确认导入

**清空历史：**
1. 打开设置 → 数据管理
2. 点击「清空数据」
3. 选择是否保留置顶记录
4. 确认清空

| 快捷键 | 功能 |

可在设置中修改全局快捷键：
1. 打开设置
2. 找到「快捷键」部分
3. 点击快捷键右侧的输入框
4. 按下新的快捷键组合
5. 自动保存

## 常见问题

### Q1: 软件是免费的吗？

小安剪贴板有免费版和 Pro 版：
- **免费版**：基础记录 + 搜索功能，无限制使用
- **Pro 版**：29.9 元一次买断，包含加密、无限历史、终身更新

### Q2: 我的数据会上传到云端吗？

**不会！** 小安剪贴板 100% 本地存储：
- 所有数据存储在您的电脑本地
- 不联网、不上传、不收集
- 离线也能正常使用

### Q3: 支持哪些操作系统？

- **Windows**: Windows 10/11 (64 位)
- **macOS**: macOS 10.15+ (Intel 和 Apple Silicon)
- **Linux**: Ubuntu 18.04+、Debian 10+、Fedora 32+ 等主流发行版

### Q4: 激活码可以在多台电脑使用吗？

根据您购买的版本：
- **标准版**：1 台设备
- **家庭版**：3 台设备
- **企业版**：5 台设备

如需更换设备，请先在原设备反激活。

### Q5: 忘记了密码怎么办？

出于安全考虑，密码无法找回。但您可以：
1. 卸载软件（会清除所有数据）
2. 重新安装
3. 设置新密码

**建议：** 将密码记录在安全的地方（如密码管理器）。

### Q6: 软件占用多少资源？

小安剪贴板非常轻量：
- **安装包**：约 10-15 MB
- **内存占用**：约 30-50 MB
- **CPU 占用**：后台监听 <1%

### Q7: 如何设置开机自启？

1. 打开设置
2. 开启「开机自启」选项
3. 软件会自动添加到系统启动项

### Q8: 为什么图片无法显示？

可能原因：
1. 图片文件被删除（存储在本地）
2. 权限问题（Linux 常见）
3. 图片格式不支持

解决方法：
- 检查图片文件是否存在
- 确保软件有读取权限
- 重新复制图片

## 附录

### 数据存储位置

**Windows:**
```
C:\Users\<用户名>\AppData\Local\ClipZen\
```

**macOS:**
```
~/Library/Application Support/ClipZen/
```

**Linux:**
```
~/.local/share/ClipZen/
```

### 系统要求

**最低配置：**
- CPU: 双核 1.5 GHz
- 内存：2 GB RAM
- 硬盘：100 MB 可用空间
- 系统：Windows 10 / macOS 10.15 / Linux 4.4+

**推荐配置：**
- CPU: 四核 2.0 GHz+
- 内存：4 GB RAM
- 硬盘：500 MB 可用空间（存储大量历史记录）
