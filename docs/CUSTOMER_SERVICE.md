## 📌 下载安装指导

### Windows 用户

```
【Windows 下载安装】

1️⃣ 访问：https://github.com/lcb/ClipZen/releases/latest

2️⃣ 下载：ClipZen_0.1.0_x64-setup.exe

3️⃣ 双击运行安装程序

4️⃣ 按提示完成安装

5️⃣ 安装完成后自动启动

⚠️ 如遇 Windows SmartScreen 警告：
   点击「更多信息」→「仍要运行」
   （因为我们是新软件，还未签名）

安装遇到问题了吗？告诉我具体情况~
```

### macOS 用户

```
【macOS 下载安装】

1️⃣ 访问：https://github.com/lcb/ClipZen/releases/latest

2️⃣ 下载：
   - Apple Silicon (M1/M2): ClipZen_0.1.0_aarch64.dmg
   - Intel: ClipZen_0.1.0_x64.dmg

3️⃣ 双击打开 DMG 文件

4️⃣ 拖动 ClipZen 到「应用程序」文件夹

5️⃣ 首次运行：
   - 右键 ClipZen → 打开
   - 点击「打开」确认

⚠️ 如遇「无法打开」：
   系统偏好设置 → 安全性与隐私 → 仍要打开

需要远程协助吗？😊
```

### Linux 用户

```
【Linux 下载安装】

Ubuntu/Debian:
1️⃣ 下载：ClipZen_0.1.0_amd64.deb
2️⃣ 终端运行：sudo dpkg -i ClipZen_0.1.0_amd64.deb
3️⃣ 应用菜单启动

或使用 AppImage:
1️⃣ 下载：ClipZen_0.1.0_amd64.AppImage
2️⃣ 终端运行：chmod +x ClipZen_0.1.0_amd64.AppImage
3️⃣ 运行：./ClipZen_0.1.0_amd64.AppImage

Fedora:
1️⃣ 下载：ClipZen_0.1.0_amd64.rpm
2️⃣ 终端运行：sudo dnf install ClipZen_0.1.0_amd64.rpm

遇到问题了吗？告诉我具体错误信息~
```

## 💡 客服技巧

### 响应时间
- 首次响应：< 1 分钟（自动回复）
- 人工响应：< 5 分钟（工作时间）
- 问题解决：< 24 小时

### 语气风格
- 友好亲切，用 emoji 增加温度
- 专业但不生硬
- 主动询问，不要等用户追问

### 问题分类
1. **售前咨询** → 产品介绍、功能说明
2. **售中指导** → 下载、安装、激活
3. **售后支持** → 技术问题、退款
4. **用户反馈** → 建议、bug 报告

### 升级处理
遇到以下情况及时升级：
- 用户情绪激动 → 先安抚，再解决
- 技术难题 → 记录详细信息，转技术
- 批量问题 → 记录共性问题，统一处理
